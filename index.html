05 — Qualitative Improvement Themes

Six convergent themes from instructor feedback — independent of format

Instructors provided written improvement suggestions alongside their preference votes. Across 308 total extracted improvement points, six recurring themes emerge that apply to both options and both subjects — revealing universal design gaps the chatbot must address regardless of which format is chosen.

1Concrete Manipulatives Must Be Specified by Name, Every Time
Both OptionsMathsLanguage

The most frequently cited gap across all four data segments. Instructors want explicit, named manipulative instructions — tilli bundles, number cards, counting charts, picture cards, finger tracking — written directly into each response. Both options describe what to teach without consistently specifying with what. Maths instructors specifically request that tilli-bundle steps not be omitted "because teachers already know" — they don't always, especially on specific days. Language instructors request picture cards and visual prompts for story activities. This is a mandatory content element, not an optional detail.

2Camp-Day Context Must Drive Activity Calibration
LanguageBoth Options

Language instructors repeatedly noted that early-camp (Days 1–10) and late-camp (Days 15–25) contexts require fundamentally different responses. By Day 20, children are at paragraph/story level — responses must not prescribe barahkhadi drill or beginner-level phonological tasks. The Day 20 phonological question (Q12) — the second-largest win margin in the dataset — demonstrates this vividly: Option 1 lost heavily because it prescribed Day 1–5 activities for Day 20. Both options need a camp-day parameter embedded in Language prompts, creating at least three response variants: early (Days 1–10), mid (Days 11–18), and late (Days 19–25).

3Word Problems as Default Format for All Maths Operations
MathsBoth Options

Instructors from both Maths segments independently requested that mathematical tasks be framed as contextualised word problems rather than bare expressions. Shopkeeper-customer, market, calendar, and school scenarios were specifically named. One instructor articulated it directly: "if I say '25 minus 12, solve it' the child gives an answer — but if I say 'there were 25 mangoes, 12 were sold, how many remain?' the child also learns language, context, and reasoning." Both options under-deliver on this. Word problem format should be the default for any operational Maths response, with raw expressions only as a fallback.

4Level Labels Without Differentiated Tasks Are Useless
Both OptionsLanguageMaths

Both options list learner groups by name (Beginner, Letter, Word, NR1, NR2) but often prescribe the same or similar activities for all. Language instructors asked for distinct tasks for observers, letter-level, word-level, and story-level children in the same response — not just level labels on one activity. Maths instructors requested genuinely differentiated problems by level (15-5 for beginners; 63-27 for NR2; 234-76 for Sub/Div), not just the same tilli-bundle task with a note to simplify. Labelling levels without differentiating their tasks is a design half-measure.

5Peer Learning Scaffolds Are Universally Missing
Both OptionsLanguageMaths

Both Language and Maths improvement lists independently surface peer explanation mechanisms — having one child demonstrate a solution to the group, organising 4–5 child retelling groups, using advanced learners to support peers. Neither option routinely includes a "peer task" scaffold. This is a low-cost, high-impact enhancement: one sentence per response recommending a peer explanation moment would close this gap and align with TaRL's group-learning pedagogy. It requires no format change — just a standardised closing sentence.

6Language Responses Need Phonological Sequencing Awareness
Language Only

Language instructors raised a distinct set of gaps not present in Maths feedback: explicit sequencing for vowel marker (matra) instruction, complex word formation using 2–3 syllables after barahkhadi mastery, and the distinction between observation-level children vs. letter-level children in the same class. A specific request appeared in multiple responses: Language phonological activities should know whether children are still at the barahkhadi stage or have moved past it — and calibrate accordingly. This is the camp-day awareness issue applied specifically to phonological progression.

06 — Product & Content Recommendations

Six priority recommendations for chatbot response design

Ordered by impact and implementation feasibility. Recommendations 1–3 are structural changes to the response system; 4–6 are content enhancements applicable to both format options.

Do not deploy one default format. Route Maths questions to Option 1's structured scaffolded format. Route Language questions based on intent: Option 1 for level-differentiated activity design; Option 2's action-first format for lesson planning, content delivery, and broad activity suggestions. An intent classifier by subject + question type enables this routing with minimal engineering.

High Priority

Every response — regardless of format — must include a named materials step. "Use tilli bundles and number cards" must appear explicitly, not be assumed. Template this as a required response element that is never omitted. The single most-common improvement request across all four data segments. Language responses must detect the camp day from the teacher's query and route to one of three response variants: early (Days 1–10), mid (Days 11–18), late (Days 19–25). This applies to phonological, story, and content activities.

Market, shopkeeper-customer, calendar, and school scenarios are specifically requested. 05

Add one standardised sentence to every small-group activity response recommending a peer explanation moment: "Ask one child to show their solution to the group using the tilli bundles" or "Pair a story-level child with a letter-level child for re-reading." 06
Exploratory

Scale the Survey with Rubric Scoring

The current survey has 16 question pairs with 141 total votes — robust directionally but thin for some categories (Content by Level has only 5 total votes). Expand to 30+ question pairs with 8+ votes each. Add structured rubric scoring alongside binary preference: clarity, pedagogical accuracy, cultural fit, level differentiation quality, and actionability. This would enable statistically significant and category-specific prompt fine-tuning.

07 — Prompt Design Guidance

The survey result is not a binary choice between Option 1 and Option 2. The evidence points to a hybrid, context-adaptive architecture where Option 1's reasoning depth is combined with Option 2's action-first delivery style, deployed based on subject and question intent.

Recommended architecture: Use Option 1 as the primary format for Maths and level-differentiated Language questions. Use Option 2's action-first, numbered-sequence style for Language lesson planning, broad activity suggestions, and content delivery. 08 — Conclusion

Option 1 for Maths. Context-adaptive for Language. Universal content fixes for both.

A concise synthesis of the survey findings and the clear path forward for the TaRL chatbot response design.

What the survey tells us

Option 1's structured, principle-based format leads overall with 53.9% of 141 votes. But this headline masks a critical subject split: Maths instructors prefer Option 1 at 62.9% while Language instructors prefer Option 2 at 53.2%. The overall lead for Option 1 is entirely produced by Maths. At the question level, wins are perfectly split 7-7 with 2 ties — confirming that neither format has a universal structural advantage.

The deepest insight is this: instructor preference is determined by question type, not format alone. When a question inherently requires scaffolding, progression, or multi-level differentiation — Option 1 wins. When a question calls for an immediately executable action sequence — Option 2 wins. The right format is a function of the question intent, not a global setting.

Below this format question, six universal content gaps persist regardless of format: missing manipulative specifications, camp-day context blindness in Language, absence of word problem framing in Maths, level labels without genuine task differentiation, no peer learning scaffolds, and phonological sequencing gaps in Language. These must be addressed in both options.

The path forward

Deploy Option 1 as the default for all Maths queries and for Language questions that explicitly request level differentiation. Deploy Option 2's direct, numbered-sequence style for Language lesson planning, broad activity suggestions, and content delivery. Fix the six universal content gaps in both formats. Add camp-day detection to all Language prompts. Mandate manipulative specification and peer learning scaffolds in every response. The result will be a chatbot that responds the way each teacher's question actually demands — not one that applies a single global format to every context.
