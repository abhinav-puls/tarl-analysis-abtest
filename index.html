04 — Qualitative Improvement Analysis

Six convergent themes emerge from 256 instructor improvement suggestions

Instructors provided open feedback alongside their vote for both response options. LLM-extracted improvements — 74 for Language Opt 1, 71 for Language Opt 2, 63 for Maths Opt 1, and 48 for Maths Opt 2 — converge on six recurring design gaps that apply universally across format and subject.

1 Concrete Manipulatives Must Be Specified, Not Assumed
Both Options Maths Language

The most frequently requested improvement across all four segments. Instructors want explicit, named manipulative steps — tally bundles, number cards, counting charts, picture cards, finger-tracking — written into each response. Both options describe what to teach without specifying with what. Maths instructors specifically asked that tally-bundle and number-card instructions not be omitted on Day 20 on the assumption that teachers already know, while Language instructors requested picture cards and visual prompts for story activities.

2 Simple-to-Complex Progression Is Expected, Not Optional
Opt 1 Strength Maths

Maths instructors repeatedly validated and requested explicit progression ladders: single-digit before two-digit, no borrowing before borrowing, counting before operations. Option 1 partially delivers this through its principle-based structure, which is likely why it wins strongly in Maths skill categories. However, instructors also flagged that the progression should be visible as a numbered sequence within the response, not embedded in prose. Feedback for Option 2 consistently asked it to "add a progression" — the clearest signal for its Maths weakness.

3 Word Problems and Real-World Context Are Universally Requested
Both Options Maths Language

Across Maths Option 2 and Language Option 1 feedback, the most common request was to replace bare mathematical expressions with contextualised word problems — shopkeeper-customer scenarios, market activities, calendar-based tasks. Language instructors paralleled this with requests for story-context writing prompts and picture-based discussion rather than abstract topics. Neither format consistently embeds real-world contextualisation. This is a content gap, not a format gap — and is actionable regardless of which option is chosen.

4 Level Differentiation Must Go Beyond Labelling — Activities Should Differ
Both Options Language Maths

Instructors from both subjects noted that responses name learner groups (beginner, letter-level, word-level, 10–99, subtraction-level) but often prescribe the same or similar activities. Language instructors asked for distinct tasks for observers, letter-level, word-level, and story-level children in the same response. Maths instructors requested differentiated worksheets with genuinely different problems by level. Both options are guilty of listing groups without meaningfully differentiating the task content.

5 Peer Learning and Explanatory Tasks Are Consistently Missing
Both Options Language Maths

Both Language and Maths improvement lists independently surface peer explanation mechanisms — having one child demonstrate a solution, organising groups of 4–5 for story retelling, using advanced learners to support others. Neither response option routinely includes a "peer task" scaffold. This is a low-cost, high-impact enhancement: a single sentence per response recommending a peer explanation moment would close this gap and align with TaRL's group-learning pedagogy.

6 Language-Specific: Phonological Awareness and Writing Progression Needs Attention
Language Only Opt 2 Strength

Language instructors — specifically for Option 2 — raised a distinct set of gaps not present in Maths feedback: explicit guidance on vowel marker (matra) instruction sequencing, complex word formation using 2–3 syllables, barahkhadi practice with challenging consonants, and role-play/drama activities for story comprehension. Opt 2 Language responses were critiqued for not differentiating between early-camp activities (Days 1–10) and late-camp ones (Days 15–25), where children are at higher proficiency levels. This suggests Language response templates need camp-day awareness baked in.

05 — Product & Content Recommendations

Six recommendations for response quality improvement

Ordered by impact and implementation effort. Recommendations 1–3 are high-priority structural changes; 4–6 are medium-priority content enhancements.

01
High Priority

Build a Subject-Aware Response Router

Do not deploy a single default response format across both subjects. Route Maths questions to Option 1's structured, scaffolded format by default. For Language, use a hybrid: Option 1's structure for level-based activity design, Option 2's directness for content delivery and lesson planning. Intent classification by subject + question type enables this routing.

02
High Priority

Mandate Concrete Manipulative Instructions

Every response — regardless of format — must include a "Materials" field specifying the physical tools to use (tally bundles, number cards, picture cards, counting charts). This was the single most-common improvement request across all four segments. Template this as a required response element, not an optional detail.

03
High Priority

Embed Real-World Word Problem Defaults

Responses should present mathematical tasks as contextualised word problems by default — not bare expressions. Shopkeeper-customer, market, and calendar scenarios are specifically requested. For Language, picture-based or story-based prompts should precede any writing or discussion task. This is a content generation instruction, not a format change.

04
Medium Priority

Redesign Lesson Planning Responses for Brevity

Option 2 wins Lesson Planning at 56% — instructors find Option 1's principled framing verbose during planning. Lesson plan responses should lead with a structured session template (timing → group → activity) before any rationale. If pedagogical reasoning is included, it should follow, not precede, the action plan. Apply across both subjects.

05
Medium Priority

Add a Mandatory Peer Learning Scaffold

Both improvement lists independently surface peer explanation as a gap. Add one sentence to every small-group activity response: a suggested peer task (e.g., "have one student explain their solution to the group" or "organise groups of 4–5 for retelling"). This aligns with TaRL pedagogy, requires minimal engineering, and addresses a universal instructor ask.

06
Exploratory

Introduce Camp-Day Awareness in Language Responses

Language instructors flagged that early-camp (Days 1–10) and late-camp (Days 15–25) contexts require fundamentally different responses. By Day 20, children are at paragraph/story level — responses must not prescribe beginner-level activities. Build a camp-day parameter into Language prompts and create response variants calibrated to early, mid, and late camp stages.

06 — Prompt Design Recommendations

Which format to go ahead with — and how to improve each

Based on the quantitative evidence and qualitative improvement analysis, the recommendation is not a binary choice between Option 1 and Option 2. The data points to a hybrid, context-adaptive architecture with specific prompt engineering interventions for each use case.

Recommended approach: Use Option 1 as the primary response architecture — its structured, scaffolded format wins overall and is strongly preferred in Maths. However, suppress the theoretical preamble and implement category-specific output templates that mirror Option 2's directness for Lesson Planning and Content Delivery contexts. The goal is Option 1's reasoning depth with Option 2's action-first communication style.

07 — Conclusion

Option 1 is the right foundation — improved with context-adaptive delivery

A concise summary of findings and the path forward for the TaRL chatbot nudge programme.

What the data says

Option 1's structured, principle-based format is the stronger default. It wins 58.5% of votes overall and delivers a decisive 68.4% preference in Maths — the subject where pedagogical scaffolding matters most. Instructors working with number concepts, word problems, and level-differentiated skill activities consistently prefer responses that explain why alongside what to do.

However, the Language finding is unambiguous: a 50–50 vote split tells us instructors do not favour either format for language instruction at the aggregate level. Within Language, context determines the winner — Option 1 for level-differentiated activity design, Option 2 for content delivery and planning. A one-size-fits-all deployment would consistently mismatch instructor needs in half of Language interactions.

The qualitative data adds a third dimension: neither option fully addresses what instructors actually need. Concrete manipulative guidance, real-world word problems, peer learning scaffolds, and genuine level differentiation are universal gaps — independent of response format — that must be engineered into the prompt template itself.

The recommendation

Deploy Option 1 as the core architecture. Implement a lightweight intent classifier that triggers an "action-first, rationale-second" output style for Lesson Planning and Content Delivery contexts. Apply six universal content enhancements — materials specification, word problem defaults, peer moments, progression ladders, level differentiation, and camp-day awareness — to every response regardless of format.
